== Description ==

Embeds an interactive iTunes Preview for an artist as a sidebar widget

= Plugin's Official Site =

http://noblegiant.com/blog/2010/11/itunes-preview-widget/

== Installation ==

1. Upload the entire `itunes-preview-widget` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. You will find the widget under Appearance > Widgets in the Wordpress backend. All settings are editable there for each widget instance.
